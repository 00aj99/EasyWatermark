<h1 align="center">水印</h1>

<p align="center">  
安全、快速地为你的敏感照片添加水印，防止被小人泄露、利用。
  </br>
尤其适合因不可抗力而需上传的：（手持）身份证照、隐私证件照、早期小样图片、带版权图片等照片。
</p>
</br>

<p align="center">
<img src="/static/preview.png"/>
</p>

> 当然，也适用于制作表情包，毕竟支持图片水印，效果十分鬼畜。
## 特性

> 当时我就念了一首诗 👓

- 单纯离线本地应用，代码开源想看就看
- 横竖间距均可调节，颜色明暗随心转换
- 大小角度自由旋转，文字图片皆可打上
- 水印重复全图铺满，坏蛋除水印难上难
